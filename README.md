Este é um projeto desenvolvido com um propósito muito específico e especial: convidar uma pessoa incrível para sair. Por que enviar uma mensagem comum se eu posso enviar um link, certo? A ideia aqui foi unir a paixão pela tecnologia com a vontade de criar um momento memorável.

🛠️ Tecnologias Utilizadas
Para garantir que o "Sim" venha com estilo, utilizei:

HTML5 & CSS3: Para uma interface limpa e responsiva.

JavaScript: Para a lógica dos botões (especialmente aquele botão de "Não" que insiste em fugir 😉).

Vercel/Netlify: Para hospedar o convite e torná-lo acessível em qualquer lugar.

📋 Funcionalidades
Pergunta Irrecusável: Uma interface direta ao ponto.

Botão "Não" Inteligente: Caso ela tente clicar no "Não", o botão se move aleatoriamente pela tela, provando que essa opção não é válida.

Confirmação Especial: Ao clicar no "Sim", uma animação de fogos ou corações é disparada (e talvez um redirecionamento direto para o meu WhatsApp).
